# 💬 Real-Time Messenger Platform

**ChatApp** is a full-stack web application that enables real-time messaging and communication between users. Built with modern technologies like Node.js, MongoDB, React, and TypeScript, it provides a seamless chat experience with features like real-time messaging via WebSocket, user authentication, group chats, presence tracking, and file uploads. The application features a responsive UI with light/dark mode support, making it a production-ready solution for real-time communication needs.

## 🗝️ Key Features

- ✅ Authentication with Secure Cookies
- 🔌 Real-Time Messaging via WebSocket (Socket.io)
- 💬 Create One-on-One or Group Chats
- 👥 Join & Leave Rooms in Real-Time
- 🟢 Online / Offline User Presence
- 💬 Reply to Specific Messages
- ⚡ Real-Time Last Message Updates
- 📁 File Upload
- 🌗 Light & Dark Mode
- 📱 Fully Responsive UI
- 🎨 Styled with **Tailwind v4** + **daisyUI**
- 🧩 Built with **Node.js**, **MongoDB**, **React**, and **TypeScript**
- 🚀 Deployment Ready
